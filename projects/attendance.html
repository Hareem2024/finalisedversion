Attendance Detection System

Tech Stack: Python, face_recognition, OpenCV, CSV logging

A simple but effective attendance system using facial recognition — useful for classrooms, labs, or secure spaces.

  • Implemented with face_recognition and OpenCV for live webcam face matching
  • Logs names and timestamps to a local CSV
  • Prevents duplicate entries and runs fully offline

Why it matters: A practical automation project that touches backend processing, user data logging, and light ML — with zero cloud dependency.
