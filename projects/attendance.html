Attendance Detection System

Tech Stack: React, TypeScript, Tailwind CSS, face-api.js, ES Modules

A modern browser-based attendance system using facial recognition technology, built with React and TypeScript for robust development.

  • Implemented face detection and recognition using face-api.js with TensorFlow.js models
  • Built with React and TypeScript for type safety and maintainable code
  • Created responsive UI with Tailwind CSS for modern design
  • Uses ES Modules for modern JavaScript module system
  • Persists registered user data using LocalStorage

Why it matters: Demonstrates the power of modern web technologies in creating sophisticated applications that run entirely in the browser, with no server requirements and strong type safety.
