Markdown Editor

Tech Stack: React, TypeScript, Y.js (CRDT), WebRTC, IndexedDB, TailwindCSS

A privacy-first, real-time collaborative markdown editor — like Google Docs for developers — that works even offline.

  • Built with Y.js CRDTs for conflict-free editing across multiple users
  • Uses WebRTC for direct peer-to-peer syncing with no backend
  • Edits are persisted via IndexedDB and sync when back online
  • Includes user presence indicators, activity logs, and a live Markdown preview pane
  • Designed a custom dark UI with split editor/preview layout and fully responsive behavior

Why it matters: Built from scratch to explore distributed state, offline-first architecture, and collaborative UI — all without third-party APIs.
