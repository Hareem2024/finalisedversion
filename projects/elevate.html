SafeWatch Detection App

Tech Stack: JavaScript, MediaPipe, Web Audio API, Tailwind CSS

Developed a real-time drowsiness and distraction detection system using computer vision and audio analysis.

  • Implemented MediaPipe for facial landmark detection and eye tracking
  • Used Web Audio API for sound analysis and alert generation
  • Built responsive UI with Tailwind CSS for cross-device compatibility
  • Integrated real-time alerts and notifications for user safety
