SafeWatch Detection App

Tech Stack: JavaScript, MediaPipe Face Mesh, Web Audio API, TailwindCSS

A browser-based real-time app that detects user drowsiness or distraction using just the webcam — no AI APIs or cloud required.

  • Uses MediaPipe Face Mesh to detect eye closure, yawns, and head turns
  • Built a custom alert system using Web Audio API + visual overlays
  • Optimized for fully in-browser, privacy-respecting use cases
  • Designed for dashboard tablets, study/work focus, and accessibility use

Why it matters: Proves real-world computer vision UX can run fully offline with open-source models — built for actual safety and focus enhancement.
